## 📁 Folder Structure

- Each topic has its own folder.
- Solution files (e.g., `.cpp`, `.py`, `.java`) are added directly inside the relevant topic folder.
- File names should clearly indicate the problem, like:
  - `reverse_linked_list.cpp`
  - `two_sum.py`
  - `longest_substring_without_repeating_characters.java`

**Example:**
```
03_arrays/
  ├── two_sum.py
  ├── best_time_to_buy_and_sell_stock.cpp
```
